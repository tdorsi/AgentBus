# Agent Rules

- Read `sprint.md`, `tasks/active.md`, `tasks/blocked.md`, and your inbox before starting.
- Claim only one task at a time.
- Use task IDs for all task-related work.
- Use message IDs for all cross-agent messages.
- Append to your own log.
- Do not overwrite another agent's log.
- Do not delete history.
- Record durable decisions in `decisions/decision_log.md`.
- Ask for human review when blocked, uncertain, or changing shared workflow.
- Keep updates concise.
- Preserve existing files and append notes instead of overwriting content.
